# To Do List

## Description
-_Allows a user to manage tasks by adding them and checking off done tasks_

### By _**Austin Minnon & Chris Kuiper**_

## Setup & Installation

-* _Clone this repository._
 -* _Run the following commands in terminal:_
 -  * _npm install_
 -  * _bower install_
 -  * _gulp build_
 -  * _gulp serve_
